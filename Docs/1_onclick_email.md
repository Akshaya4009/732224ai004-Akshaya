- create a new workflow
- Add a manual trigger.
- Next, add the Gmail node, which will send an email to the specified address.
- Connect the manual trigger to the Gmail node.
- Open the Gmail node, which contains fields for the recipient email address, subject, and message.
- Add the Gmail credentials, and then execute the workflow.
- After execution, the email will be sent to your Gmail account, and the process will be completed.
